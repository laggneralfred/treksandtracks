	Fixed width issue when switching to different themes with Featured Images
	Removed the following filters:
		it_featured_images_display_height_textbox
		it_featured_images_filter_height filter
		it_featured_images_filter_width_height
	Returning any value other than true to the filter it_featured_images_text_overlay
		will disable the Text Overlay feature and hide the options
	Added two new filters:
		it_featured_images_filter_allow_variable_width
			By default, this is false. Return a non-false value to allow the database
			to control the width
		it_featured_images_filter_allow_variable_height
			By defauilt, this is true. Return false to allow the database to control
			the height
	Fixed output JavaScript code to allow Featured Images to work from inside
		iFrames. This fixes the problem where Featured Images wouldn't render
		inside theme previews.
